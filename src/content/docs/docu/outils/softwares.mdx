Mes ordinateurs portables sont sous [CachyOS](https://cachyos.org/), un dérivé d'[Archlinux](https://archlinux.org/). J'utilise principalement [Hyprland](https://hypr.land/), sinon [Plasma](https://kde.org/fr/). Mon ordinateur de bureau est sous Windows, et mes serveurs sont sous [Debian](https://www.debian.org/index.fr.html).

## Cross-Platform

### Firefox

[Firefox](https://www.mozilla.org/fr/firefox/new/) est un navigateur sobre et efficace, basé sur Gecko. Il représente la seule alternative à Chromium et ses dérivés. Add-ons utilisés :

- [µBlockOrigin](https://ublockorigin.com/fr) : bloqueur de pub léger et efficace
- [BitWarden](https://bitwarden.com/download/) : gestionnaire de mots de passe synchronisé sur mon instance locale
- [Page-Assist](https://github.com/n4ze3m/page-assist) : dialogue avec mon instance Ollama privée
- [Ghostery](https://addons.mozilla.org/fr/firefox/addon/ghostery/) : auto-consent sur les bannières de cookies
- [ClearURL](https://addons.mozilla.org/fr/firefox/addon/clearurls) : suppression automatique des trackers URL

Navigateurs dérivés : [LibreWolf](https://librewolf.net/) (vie privée) ou [Zen Browser](https://zen-browser.app/) (productivité).

### Signal

[Signal](https://signal.org/fr/) est un système de messagerie instantanée entièrement chiffrée.

## Linux

### btop
[btop](https://github.com/aristocratos/btop) — version premium de `top` pour surveiller les ressources et processus.

### duf
[duf](https://github.com/muesli/duf) — version premium de `du` pour surveiller l'espace disque.

## Windows

### WinGetUI
[WinGetUI](https://github.com/marticliment/UniGetUI) — installer des paquets depuis plusieurs stores (Chocolatey, pip, scoop, winget).

### FlowLauncher
[FlowLauncher](https://github.com/Flow-Launcher/Flow.Launcher) — launcher et moteur de recherche remplaçant le menu démarrer.

### Files
[Files](https://github.com/files-community/Files) — remplaçant du gestionnaire de fichiers, indépendant et opensource.

### VSCodium
[VSCodium](https://vscodium.com/) — fork entièrement opensource de Visual Studio Code.

### Rufus
[Rufus](https://rufus.ie/fr/) — création d'USB bootables rapide.

### Tabby
[Tabby](https://tabby.sh/) — émulateur de terminal (telnet, ssh, powershell, cmd).

## Android

### Material Files
[Material Files](https://play.google.com/store/apps/details?id=me.zhanghai.android.files) — navigation dans les dossiers sans pubs.

### ReVanced Manager
[RVM](https://github.com/vancedapps/rv-manager) — installer des APK patchées (YouTube, Spotify).
