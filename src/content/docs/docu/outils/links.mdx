## Windows

### MAS (Microsoft Activation Scripts)
[massgrave.dev](https://massgrave.dev/) — scripts d'activation Windows & Office open-source. Méthode HWID, KMS38, TSforge.

### Chris Titus Tech — WinUtil
[WinUtil](https://github.com/ChrisTitusTech/winutil) — utilitaire PowerShell tout-en-un pour configurer Windows : désactiver la télémétrie, installer des programmes, tweaker le système.

### BCUninstaller
[BCUninstaller](https://github.com/Klocman/Bulk-Crap-Uninstaller) — désinstalleur en masse pour Windows. Détecte les résidus, supprime proprement.
