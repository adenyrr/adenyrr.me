## Ajouter les dépôts

`contrib` et `non-free` sur Debian :

```bash
sudo sed -i 's/^deb .*/& contrib non-free/g' /etc/apt/sources.list && apt update
```

:::note[La commande sed]
Une explication de la commande `sed` sur le site d'[ubuntu-fr](https://doc.ubuntu-fr.org/sed).
:::

## Installation de paquets de base

Beaucoup de LXC sont basées sur Debian. L'équilibre entre le hardening et le confort d'administration est complexe ; c'est pourquoi je préfère utiliser des images minimales avec quelques utilitaires par dessus.

:::note[su ... do ?]
La commande `sudo` permet de lancer des commandes root depuis l'utilisateur actif. Pour ajouter l'utilisateur au groupe sudoers : `usermod -aG sudo $user`
:::

```bash
apt update && apt install btop curl gpg sudo ufw
```

- **btop** : utilitaire plus joli et complet que top
- **curl** : récupérer du contenu distant
- **gpg** : clefs gpg pour signer des dépôts
- **sudo** : privilèges d'un autre utilisateur
- **ufw** : gestion de pare-feu

:::caution[ufw]
Après l'installation d'ufw, autorisez le port SSH : `ufw allow ssh` puis `ufw enable`.
:::

## Ajouter Docker et Docker Compose

```bash
curl -fsSL https://get.docker.com | sudo sh
```

Le script vérifie la version de l'OS, ajoute le dépôt et installe Docker et ses dépendances.
