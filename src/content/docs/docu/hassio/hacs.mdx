## HACS

[HACS](https://hacs.xyz/) (Home Assistant Community Store) est un store communautaire qui permet d'installer des intégrations, des thèmes et des cartes Lovelace non incluses dans le dépôt officiel de Home Assistant.

### Installation

1. Téléchargez et exécutez le script d'installation :

```bash
wget -O - https://get.hacs.xyz | bash -
```

2. Redémarrez Home Assistant
3. Allez dans **Paramètres** → **Appareils et services** → **Ajouter une intégration** → cherchez **HACS**
4. Suivez les instructions de liaison avec votre compte GitHub

### Intégrations recommandées via HACS

- **Alarmo** : système d'alarme complet
- **Browser Mod** : contrôle avancé du navigateur
- **Mushroom Cards** : cartes Lovelace modernes et élégantes
